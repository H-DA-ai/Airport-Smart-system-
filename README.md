Smart Airport Security System

 Overview

The Smart Airport Security System is an AI-powered surveillance solution that uses real-time CCTV feeds to detect and recognize faces. It identifies criminals, flags suspicious individuals, and improves airport security through automation and real-time alerts.

 Features

* Real-time face detection using OpenCV
* Face recognition using AI models
* Criminal identification (Criminal / Suspicious / Safe)
* Mask and disguise handling
* Alert system with confidence score and match distance
* Multi-camera tracking support
* Secure data handling

 Tech Stack

* Python
* OpenCV
* NumPy
* Deep Learning (Face Recognition Models)
* HTML, CSS, JavaScript (for UI)

 Project Structure

smart-airport-security/

 Dataset

The dataset is not included in this repository due to privacy reasons.
Users can create their own dataset by organizing images into labeled folders.

 How to Run

1. Install dependencies:

   ```
   pip install -r requirements.txt
   ```
2. Run the system:

   ```
   python security.py
   ```
3. Open `index.html` in your browser (if UI is included)

 Output

* Live video feed with bounding boxes
* Labels: Criminal / Suspicious / Safe
* Alerts with name, confidence score, and match distance
